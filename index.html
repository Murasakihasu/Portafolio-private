Proyectos

Maneki pymes

Maneki Pymes es una aplicación web chilena creada como parte de mi certificación de estudios, diseñada para apoyar a pequeños emprendedores.

Esta appweb ofrece información clave para iniciar y administrar un negocio según el rubro, con un acompañamiento continuo para optimizar su gestión. Además, incluye un foro con noticias, cursos y eventos para emprendedores, donde pueden compartir experiencias y consejos. Mis responsabilidades abarcaron el desarrollo del frontend con JSP, CSS y JavaScript, así como la integración con el backend usando Spring Boot, JPA, JSTL y Hibernate para la persistencia de datos. Se empleó Maven para la gestión de dependencias y automatización del proyecto.

Weddings Invitations

Sitio web dedicado a automatizar el envío de invitaciones de boda.

Los novios deseaban un servicio que les permitiera gestionar, en un panel limpio y claro, la lista de invitados y sus confirmaciones, así como compartir información importante para el día especial, como lugar, fecha, hora y detalles adicionales. Lideré el desarrollo completo del proyecto, aplicando mis conocimientos en Java fullstack. Utilicé JSP, CSS y JavaScript para el frontend, mientras que el backend fue desarrollado en Java con Spring Boot y Hibernate. Implementé MySQL como base de datos y realicé el despliegue en Railway, utilizando el control de versiones con GitHub y Git.

Exámen Cinturón

Maqueta de sitio web para un juego en donde se podían crear, eliminar, dar por rendidas y reclamar mesas de otros

El reto consistía en desarrollar una maqueta funcional en un plazo de 4 horas, incorporando validaciones en los formularios y estilos en el frontend. El código debía entregarse dentro de este límite de tiempo, sin posibilidad de cambios posteriores. Además, se contaba con 24 horas para realizar el despliegue, el cual completé usando Github y Railway. La prueba requería utilizar HTML y CSSen el frontend junto a backend en Java usando JPA, JSTL Y Hibernate para la persistencia de datos en una base de datos MYSQL. Se usó Maven para gestionar las dependencias y automatizar el proyecto, y utilicé Bootstrap para acelerar la implementación de estilos en la interfaz.
